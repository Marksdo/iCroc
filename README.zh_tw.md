croc是一個允許任何兩台計算機簡單且安全地傳輸文件和文件夾的工具。據我所知，croc是唯一一個具有以下所有功能的命令行文件傳輸工具：

- 允許任何兩台計算機通過中繼傳輸數據
- 提供端到端加密（使用PAKE）
- 支持跨平台傳輸（Windows、Linux、Mac）
- 允許多個文件傳輸
- 允許恢復中斷的傳輸
- 不需要本地服務器或端口轉發
- 優先選擇IPv6，並具有IPv4回退
- 可以使用代理，如Tor

基於這個命令行應用程序的原始碼可以在這裡找到：

https://github.com/schollz/croc
