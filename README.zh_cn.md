croc是一个工具，可以简单安全地在任意两台计算机之间传输文件和文件夹。据我所知，croc是唯一一个可以满足以下所有要求的命令行文件传输工具：

- 允许任意两台计算机传输数据（通过中继）
- 提供端到端加密（使用PAKE）
- 实现跨平台传输（Windows，Linux，Mac）
- 允许多个文件传输
- 允许恢复中断的传输
- 无需本地服务器或端口转发
- IPv6优先，降级至IPv4
- 可以使用代理，如tor

此应用基于命令行应用程序，可以在此处找到：

https://github.com/schollz/croc
